# hydra-login-consent

Golang http middleware for implementing the User Login and Consent flow of Ory OAuth2 service ([Hydra](https://github.com/ory/hydra)). 

## Example implementation

### Build

```
task build-single
```

### Edit the config file

`./config.toml`

### Run

```
task run
```
